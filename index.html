Introduction

The Buffer List shows a list of jEdit's open files and a second list with recently opened files.

MOUSE ACTIONS

On the open files list:

  • single click jumps to the selected buffer
  • double click closes the selected buffer

On the recent files list:

  • double click opens the selected buffer

KEY ACTIONS

Buffer List is fully keyboard navigable. A prerequisite is that you assigned a keyboard shortcut to the action "Bring Buffer List to front", that is available after installation of BufferList.

You can assign a keyboard shortcut to this action under "Utilities -> Global options -> Shortcuts -> Commands". I suggest "A+b".

Key Shortcut Action
[Alt-b] Show Buffer List
[Up] / [Down] Choose buffer
[Enter] / [Return] Jump to/Open selected buffer
[Tab] Switch between Open and Recent files list
[Esc] Close window (only, if not docked)

APPEARANCE

Both lists can be displayed in one-column or two-column layout.

The appearance of each entry in the buffer list depends on the state of its buffer:

  • a unmodified buffer appears with a small grey orb
  • a modified buffer appears with a small orange orb
  • a new buffer appears with a small green orb
  • a new and modified buffer appears with a small yellow orb
  • a read-only buffer appears with grey text
  • the current buffer appears with a bold font

Customization

APPEARANCE

DOCKING

Buffer List may be docked into the view.

Open "Utilities -> Global options -> Docking" to define the position, where Buffer List will be docked.

Comparison

Some of you may already have noticed that jEdit's BufferTabs and the plugins Buffer List and Buffer Selector do the same thing. Each of them adresses different issues:

  • BufferTabs are jEdit's default way of switching buffers with the mouse. Chances are, that they are no longer implemented using Tabs in future jEdit versions because of it's inherent navigational disadvantages.
  • Buffer Selector's main goal is saving screen space. It is a small toolbar attached to the top or bottom of the view and doesn't get in the way.
  • Buffer List is dockable. That means you can dock it into the current jEdit view, for example to the left or right of the text, if you got some free space there. The second advantage is, that it is fully keyboard navigable.

So, basically all three are just three different ways of adressing the same problem. Choose the one method that suits you best.
