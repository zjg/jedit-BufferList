BufferList Plugin

 By Dirk Möbius
dmoebius@gmx.net

Introduction

The BufferList Plugin shows the list of currently open files in a window, that may be docked permanently into jEdit's view. Some actions that operate on buffers, like open/save/reload, may be invoked by a popup that appears when right-clicking an entry.

Behaviour

Mouse Actions

  • single click jumps to the selected buffer
  • double click closes the selected buffer (this may be disabled in the options pane, see below)
  • shift click selects multiple entries
  • right click opens a popup with some buffer related actions

Menu Actions

The following menu actions are available in menu Plugins>Buffer List:

  • Buffer List (Toggle)
    Show/Hide the Buffer List.
  • Go to Buffer List
    Show the Buffer List and set the focus on the current buffer entry.
  • Go to Next Buffer in Buffer List
    Go to the buffer below the current one in the Buffer List. This is different from View>Go to Next Buffer, because the Buffer List might be sorted differently than jEdit's internal list.
  • Go to Previous Buffer in Buffer List
    Go to the buffer above the current one in the Buffer List.

Key Actions

The Buffer List is fully keyboard navigable.

A prerequisite is that you assigned a keyboard shortcut to the plugin action Go to Buffer List. You can assign keyboard shortcut to this action under Utilities>Global options>Shortcuts. I suggest that you use "A+b".

Key Shortcut Action
[Alt-b] Show Buffer List
[Up] / [Down] Choose buffer
[Right] Expand a directory entry
[Left] Collapse a directory entry
[Tab] Switch between Open and Recent files list
[Enter] / [Return] Jump to/Open selected buffer
[Esc] Return to text area

Customization

Appearance

You can customize the appearance of the Buffer List under Utilities>Global options>Plugin Options>Buffer List. The following options are available:

  • Font
    The base font of bufferlist entries.
    The current buffer will be shown bold. All entries will be shown with a color that matches one of the regular expressions in Utilities>Global options>File System Browser>Colors.
  • Auto-Show on buffer change
    If checked, the Buffer List is automatically shown, if the current buffer has been changed.
  • Close files on double click
    If checked, a buffer that has been double clicked is closed. Otherwise it is opened.
  • Fully expand directory tree by default
    If checked, the BufferList tree will be fully expanded when first loaded.

Docking

Buffer List may be docked anywhere into the view.

Open Utilities>Global options>Docking to define the position, where the Buffer List will be docked.
