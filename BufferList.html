Introduction

The Buffer List shows the lists of currently open files and the list of recent files in a window, that may be docked permanently into jEdit's view. The lists are displayed in a table fashion style.

Release Notes

Help!!! Where's the Session Manager?!?

As of version 0.7, the session management functions have been taken out into a separate plugin called Sessions. You can download it using the Plugin Manager.

Note: a side effect of splitting the Session Manager out of Buffer List is that it can no longer be shown as BufferList's toolbar. Sorry.

The Buffer List

The BufferList shows:

  • Status icon
  • Filename
  • Directory
  • Mode name

This can be configured to some extent in the Global Options. See Customization.

Mouse Actions

On the open files list:

  • single click jumps to the selected buffer
  • double click closes the selected buffer (this may be disabled in the options pane, see below)
  • right mouse button click opens a popup with some buffer related actions (goto, open in new view, close, save, save as, reload)

On the recent files list:

  • double click opens the selected buffer
  • right mouse button click opens a popup with some file related actions (open, open in new view)

On both lists:

  • If the column headers are displayed (see Customization) single-clicking a column header sorts the list by this by this column, in ascending order.
    Clicking it again sorts the list in descending order.
    Clicking it again turns sorting off and reverts to jEdit's sort order.
  • If the column headers are displayed, double-clicking on a column header edge auto-resizes the column to the size of the largest cell in that column.
    Note: This only works if auto-resizing of columns is turned off in BufferList's options.

Menu Actions

The following menu actions are available in menu Plugins>Buffer List:

  • Buffer List
    Show the Buffer List.
  • Go to Next Buffer in Buffer List
    Go to the buffer below the current one in the Buffer List. This is different from View>Go to Next Buffer, because the Buffer List might be sorted differently than jEdit's internal list.
  • Go to Previous Buffer in Buffer List
    Go to the buffer above the current one in the Buffer List.

Key Actions

Buffer List is fully keyboard navigable. A prerequisite is that you assigned a keyboard shortcut to the plugin action Buffer List.

You can assign a keyboard shortcut to this action under Utilities>Global options>Shortcuts. I suggest "A+b".

Key Shortcut Action
[Alt-b] Show Buffer List
[Up] / [Down] Choose buffer
[Enter] / [Return] Jump to/Open selected buffer
[Tab] Switch between Open and Recent files list
[Esc] Return to text area

Customization

Appearance

You can customize the appearance of the Buffer List under Utilities>Global options>Plugin Options>Buffer List. The following options are available:

  • Auto-Show on buffer change
    If enabled, the Buffer List is automatically shown, if the buffer has been changed.
  • Show vertical lines
  • Show horizontal lines
  • Show column headers
    If the column headers are not shown, columns will be auto-resized.
  • Auto-Resize columns
    If checked, the columns are auto-resized to the width of their longest entry. You cannot resize the columns manually any longer, if this option is checked.
  • Show absolute pathname in filename column
    The filename column is always shown. Use this option, if you want to disable the directory column, but still see the directory.

Docking

Buffer List may be docked anywhere into the view.

Open Utilities>Global options>Docking to define the position, where Buffer List will be docked.

If Buffer List is docked to the left or right, the open files list is displayed above the recent files list. Otherwise they are displayed side by side.
